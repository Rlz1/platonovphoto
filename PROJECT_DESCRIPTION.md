# Platonov Photo - Веб-сайт фотографа

## Описание проекта

**Platonov Photo** — современный веб-сайт для профессионального фотографа Дениса Платонова. Сайт представляет собой полноценную веб-платформу с портфолио, услугами, системой бронирования и личным кабинетом для клиентов.

### Основные возможности

#### Публичные страницы:
- **Главная страница** — приветственный экран с избранными работами, преимуществами и отзывами
- **Портфолио** — галерея работ с фильтрацией по категориям (Свадьба, Портрет, Семья, Бизнес)
- **Услуги** — тарифы на фотосъемку с описанием и ценами
- **О фотографе** — информация о фотографе с фотографией и биографией
- **Контакты** — форма обратной связи и контактная информация

#### Система авторизации:
- **Регистрация** — создание нового аккаунта клиента (данные хранятся в localStorage)
- **Вход** — авторизация по email и паролю
- **Сессия** — сохранение состояния авторизации между перезагрузками страницы

#### Личный кабинет клиента:
- Просмотр своих фотосессий
- Статус оплаты работ
- Предпросмотр и скачивание фотографий
- Информация о дате и месте съемки

#### Панель администратора:
- Дашборд со статистикой (графики с использованием Recharts)
- Управление портфолио (добавление/удаление работ)
- Просмотр списка клиентов

## Технологический стек

### Языки программирования:
- **TypeScript** — основной язык разработки с статической типизацией
- **JavaScript** — используется в конфигурационных файлах

### Фреймворки и библиотеки:

#### Frontend:
- **React 18** — UI-фреймворк для построения интерфейса
- **React Router DOM 6** — маршрутизация между страницами

#### Стилизация:
- **Tailwind CSS 3** — utility-first CSS-фреймворк для стилизации
- **PostCSS** — обработка CSS с автопрефиксами

#### Иконки и графики:
- **Lucide React** — библиотека иконок (Camera, Instagram, Facebook, Twitter, Star и др.)
- **Recharts** — библиотека для построения графиков в админ-панели

#### Сборка и разработка:
- **Vite 5** — быстрый сборщик и dev-сервер
- **TypeScript 5** — компилятор и типизация

## Структура проекта

```
app/
├── App.tsx                 # Главный компонент с маршрутизацией
├── AuthContext.tsx         # Контекст авторизации (регистрация/вход)
├── index.tsx               # Точка входа приложения
├── index.html              # HTML-шаблон
├── types.ts                # TypeScript типы (User, PortfolioItem, Service, ClientWork)
├── mockDb.ts               # Моковые данные (портфолио, услуги, работы клиентов)
├── components/
│   ├── Header.tsx          # Шапка сайта с навигацией
│   └── Footer.tsx          # Подвал сайта
├── pages/
│   ├── Home.tsx            # Главная страница
│   ├── Portfolio.tsx       # Галерея работ
│   ├── Services.tsx        # Услуги и цены
│   ├── About.tsx           # О фотографе
│   ├── Contact.tsx         # Контакты
│   ├── Login.tsx           # Страница входа/регистрации
│   ├── ClientProfile.tsx   # Личный кабинет клиента

├── tailwind.config.js      # Конфигурация Tailwind CSS
├── postcss.config.js       # Конфигурация PostCSS
├── tsconfig.json           # Конфигурация TypeScript
└── vite.config.ts          # Конфигурация Vite
```

## Хранение данных

Все данные хранятся локально в браузере:

- **`platonov_users`** — зарегистрированные пользователи (JSON)
- **`platonov_current_user`** — текущая сессия пользователя (JSON)

## Дизайн

- Минималистичный стиль с акцентом на фотографии
- Цветовая палитра: каменные оттенки (stone) с терракотовыми акцентами (terracotta)
- Адаптивный дизайн для всех устройств
- Плавные анимации появления элементов
- Шрифт с засечками для заголовков (font-serif)

## Категории портфолио

1. **Свадьба** — свадебная съемка, love story
2. **Портрет** — индивидуальные и студийные портреты
3. **Семья** — семейные фотосессии
4. **Бизнес** — деловые портреты, корпоративная съемка
